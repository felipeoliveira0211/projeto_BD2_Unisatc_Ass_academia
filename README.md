# projeto_BD2_Unisatc_Ass_Academia
Projeto Sistema de Atividades Especiais - GRUPO A
Este projeto consiste na criação de um modelo físico de um banco de dados para um sistema de gestão para Academias. O projeto foi desenvolvido em SQL, criando um CRUD para cadastro dos itens xyz, onde a aplicacao faz conexão com o banco de dados através de uma biblioteca chamada entity framework e realiza operações de adição, aualização e deleção dos dados, usando todos os conceitos e dinâmicas práticas de SQL desenvolvidas em aula.
