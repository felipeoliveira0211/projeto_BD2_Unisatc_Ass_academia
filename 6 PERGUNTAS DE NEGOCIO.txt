Pergunta 1
Quais funcionários venderam mais assinaturas e qual a taxa de cancelamento das vendas feitas por eles?
Objetivo: avaliar desempenho comercial e qualidade das vendas.

Usa: JOIN, subconsulta, GROUP BY

SELECT
    f.id AS id_funcionario,
    f.nome_completo AS funcionario,
    c.nome AS cargo,
    COUNT(ass.id) AS total_vendas,
    SUM(CASE WHEN ass.status_assinatura = 'I' THEN 1 ELSE 0 END) AS vendas_canceladas,
    CAST(
        100.0 * SUM(CASE WHEN ass.status_assinatura = 'I' THEN 1 ELSE 0 END)
        / COUNT(ass.id)
        AS DECIMAL(5,2)
    ) AS percentual_cancelamento,
    (
        SELECT COUNT(*)
        FROM assinaturas ass2
        WHERE ass2.id_funcionario_venda = f.id
          AND ass2.renovacao_automatica = 1
    ) AS vendas_com_renovacao_automatica
FROM funcionarios f
INNER JOIN cargos c
    ON c.id = f.id_cargo
INNER JOIN assinaturas ass
    ON ass.id_funcionario_venda = f.id
GROUP BY
    f.id,
    f.nome_completo,
    c.nome
ORDER BY total_vendas DESC;

Pergunta 2
Quais planos geram maior faturamento e qual o percentual de inadimplência por plano?
Objetivo: comparar planos por receita e risco financeiro.

Usa: JOIN, agregação, CASE

SELECT
    p.nome AS plano,
    COUNT(DISTINCT ass.id) AS total_assinaturas,
    SUM(CASE 
            WHEN fp.status_pagamento = 'C' THEN fp.valor_pago 
            ELSE 0 
        END) AS valor_recebido,
    SUM(CASE 
            WHEN fp.status_pagamento = 'P' THEN fp.valor_fatura + fp.multa_aplicada 
            ELSE 0 
        END) AS valor_pendente,
    CAST(
        100.0 * SUM(CASE WHEN fp.status_pagamento = 'P' THEN 1 ELSE 0 END)
        / COUNT(fp.id)
        AS DECIMAL(5,2)
    ) AS percentual_faturas_pendentes
FROM planos p
INNER JOIN assinaturas ass
    ON ass.id_plano = p.id
INNER JOIN faturas_pagamento fp
    ON fp.id_assinatura = ass.id
GROUP BY p.nome
ORDER BY valor_recebido DESC;

Pergunta 3
Quais são os alunos mais frequentes no mês e qual posição eles ocupam no ranking?
Objetivo: identificar alunos mais engajados.

Usa: CTE, JOIN, função de janela RANK

WITH frequencia_mensal AS (
    SELECT
        a.id AS id_aluno,
        a.nome_completo,
        p.nome AS plano,
        COUNT(ac.id) AS total_acessos
    FROM alunos a
    INNER JOIN assinaturas ass
        ON ass.id_aluno = a.id
    INNER JOIN planos p
        ON p.id = ass.id_plano
    INNER JOIN acessos_catraca ac
        ON ac.id_aluno = a.id
    WHERE ac.sentido_acesso = 'E'
      AND ac.tipo_acesso = 'A'
      AND MONTH(ac.data_hora_entrada) = 1
      AND YEAR(ac.data_hora_entrada) = 2026
      AND ass.status_assinatura = 'A'
    GROUP BY
        a.id,
        a.nome_completo,
        p.nome
)
SELECT
    id_aluno,
    nome_completo,
    plano,
    total_acessos,
    RANK() OVER (ORDER BY total_acessos DESC) AS posicao_ranking
FROM frequencia_mensal
ORDER BY posicao_ranking;
